# 2주차 정리

---

## 1. Linux 기본 개념

* Linux는 **다중 사용자(Multi-user), 다중 작업(Multi-tasking)** 운영체제이다
* 모든 것은 **파일(File)**로 관리된다
* 시스템은 **계층적인 디렉터리 구조**를 가진다

---

## 2. 디렉터리 구조

* Linux는 하나의 루트 디렉터리(`/`)를 기준으로 모든 파일이 연결된다

### 주요 디렉터리 기호

```bash
.   : 현재 디렉터리
..  : 부모디렉터리
~   : 사용자 홈 디렉터리
/   : 루트 디렉터리
```

---

## 3. 경로(Path)

### 절대 경로 (Absolute Path)

* `/`부터 시작하는 경로
* 예: `/home/user/file.txt`

### 상대 경로 (Relative Path)

* 현재 위치 기준 경로
* 예: `../file.txt`

---

## 4. 기본 명령어

### 현재 위치 확인

```bash
pwd
```

* 현재 작업 디렉터리 출력

---

### 디렉터리 이동

```bash
cd 디렉터리
cd ~
cd ..
```

* `cd` : 디렉터리 이동

---

### 파일 및 디렉터리 목록 확인

```bash
ls
ls -l
ls -al
ll
```

* `ls` : 목록 출력
* `-l` : 상세 정보
* `-a` : 숨김 파일 포함
* `-al` : 모두 출력

---

### alias

```bash
alias
```

* 명령어에 별칭 설정
* 예: `ll = ls -al`

---

## 5. 파일 확인 명령어

```bash
cat 파일
more 파일
head 파일
tail 파일
wc 파일
```

* `cat` : 전체 출력
* `more` : 페이지 단위 출력
* `head` : 앞부분 출력
* `tail` : 뒷부분 출력
* `wc` : 줄/단어/문자 수 출력

---

## 6. 파일 조작 명령어

```bash
cp 파일1 파일2
mv 파일1 파일2
rm 파일
```

* `cp` : 복사
* `mv` : 이동 및 이름 변경
* `rm` : 삭제

---

## 7. 사용자 및 패키지 관리

```bash
passwd root
apt-get update
apt install gcc
```

* `passwd` : 사용자 비밀번호 변경
* `apt-get update` : 패키지 목록 최신화
* `apt install` : 프로그램 설치

---

## 8. 도움말 확인 명령어

```bash
man 명령어
info 명령어
명령어 --help
```

### 특징

* `man` : 기본 매뉴얼 (가장 많이 사용)
* `info` : 더 상세한 설명
* `--help` : 간단한 옵션 설명

---

## 9. 입출력 재지정 (Redirection)

```bash
명령어 > 파일
명령어 >> 파일
명령어 < 파일
```

* `>` : 출력 덮어쓰기
* `>>` : 출력 추가
* `<` : 입력 변경

---

## 10. 파이프 (Pipe)

```bash
명령어1 | 명령어2
```

* 앞 명령어의 출력 → 뒤 명령어의 입력으로 전달

---

## 11. Linux 명령어 사용 특징

* 명령어는 **옵션(option)**을 통해 기능을 확장한다
* 명령어 형식:

```bash
명령어 [옵션] [대상]
```
