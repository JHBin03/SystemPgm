# 디렉터리 chap2.pdf 내용 6번부터
* 모든 디바이스도 파일로 인식해서 처리한다. (디렉터리도 파일이라 생각함)

## 특별한 디렉터리 몇 가지(외워야 할 것)
```
. - 자신에 대한 정보
.. - 부모에 대한 정보
/ - 모든 디렉터리에 출발점(root), 절대경로에 대해서)(절대 경로와 상대 경로)
~ - 홈 디렉터
```
```
cat > (파일이름).txt 작성 후 컨트롤 + D
cat (파일이름).txt - 그 파일 내용 확인 가능
touch (파일이름).txt - 빈 파일 만듬. 기존에 있는 파일 이름을 쓰면 그 파일의 타임스태프를 바꿈.
```

## 파일은 두 가지로 구분된다.
1. 파일에 대한 정보
2. 실제 파일의 내용

## 경로명
* 절대 경로명 - 루트 디렉터리로부터 시작하여 경로 이름을 정확하게 적는 것
* 상대 경로명 - 현재 작업 디렉터리부터 시작해서 경로 이름을 적는 것
