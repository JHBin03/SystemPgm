# 3주차 정리

## 1. 리눅스 기본 개념

* 리눅스는 모든 것을 파일로 취급한다
* 디렉터리, 장치도 파일처럼 처리한다 
* 파일은 두 부분으로 구성된다

  1. 파일 정보 (메타데이터)
  2. 실제 데이터 

---

## 2. 기본 명령어

date → 날짜
hostname → 시스템 이름
uname → OS 정보
uname -a → 상세 정보

whoami → 현재 사용자
who → 로그인 사용자

ls → 디렉터리 목록
passwd → 비밀번호 변경
clear → 화면 초기화

---

## 3. 명령어 도움말

man → 상세 설명
info → 추가 설명
--help → 간단 사용법
whatis → 한 줄 설명 

---

## 4. 디렉터리 구조

* 리눅스는 루트(/)부터 시작하는 트리 구조이다 

[주요 디렉터리]
/ → 루트
/home → 사용자
/bin → 기본 명령어
/etc → 설정 파일
/dev → 장치
/tmp → 임시 파일

---

## 5. 특수 디렉터리

. → 현재 디렉터리
.. → 부모 디렉터리
/ → 루트
~ → 홈 디렉터리 

---

## 6. 경로(Path)

절대 경로
→ 루트(/)부터 시작

상대 경로
→ 현재 디렉터리 기준 

---

## 7. 디렉터리 명령어

pwd → 현재 위치 출력

cd → 디렉터리 이동
cd ~ → 홈
cd .. → 부모

mkdir → 디렉터리 생성
mkdir -p → 중간 경로 생성

rmdir → 디렉터리 삭제 (빈 것만 가능)

---

## 8. ls 명령어

ls → 목록 출력

옵션
-a → 숨김 포함
-l → 상세 정보
-s → 크기
-F → 파일 종류
-R → 하위 포함 

---

## 9. 파일 관련 명령어

cat → 파일 출력
more → 페이지 출력
head → 앞부분
tail → 뒷부분
wc → 줄/단어/문자 수

cp → 복사
mv → 이동/이름 변경
rm → 삭제
rm -r → 디렉터리 전체 삭제 (위험) 

---

## 10. 파일 생성

cat > 파일 → 입력 저장 (Ctrl + D 종료)

touch 파일 → 빈 파일 생성 또는 시간 변경 

---

## 11. 파일 속성

파일 속성 구성

* 이름
* 타입
* 크기
* 소유자
* 권한
* 수정 시간 

---

## 12. 파일 권한 (중요)

구성

* 사용자(u) / 그룹(g) / 기타(o)

권한

* r (읽기)
* w (쓰기)
* x (실행)

예
rw-r--r--

8진수 표현
777 → rwxrwxrwx
755 → rwxr-xr-x
644 → rw-r--r-- 

---

## 13. 권한 변경

chmod → 권한 변경

예
chmod 644 파일

기호 방식
chmod u+x 파일

---

## 14. 소유자 / 그룹

chown → 소유자 변경
chgrp → 그룹 변경 

---

## 15. 링크

하드 링크

* 같은 inode 공유
* 같은 파일

심볼릭 링크

* 다른 파일을 가리키는 파일 

---

## 16. 출력 재지정

> → 덮어쓰기
>
> > → 추가

예
who > names.txt
ls > list.txt 

---

## 17. 오류 종류

Syntax Error → 문법 오류
Runtime Error → 실행 오류
Semantic Error → 의미 오류

## 19. 함수

형식

* return 값은 타입과 일치해야 한다

## 20. 배열

## 21. 포인터

포인터 = 주소 저장 변수

int *p
p = &a

*p → 값 접근

특징

* 변수 주소를 통해 값 변경 가능 

---

## 22. 배열과 포인터

배열 이름 = 주소

p = a 가능
