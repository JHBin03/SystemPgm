## 시스템 호출과 C 라이브러리 함수
1. 시스템호출(System Calls)
  * 운영체제의 핵심을 '커널'이라고 함
  * 커널에 있는 기능을 쓰는 것.
  * C 함수처럼 호출 됨.
2. C 라이브러리 함수(Library Functions)
  * C 라이브러리함수는보통시스템호출을포장해놓은함수
  * 보통 내부에서 시스템을 호출
  * ANSI C Standard Library

## 시스템 호출
1. 시스템호출은 커널에서 비스요청을 위한 프로그래밍 인터페이스
2. 응용프로그램은 시스템호출을 통해서 커널에 서비스를 요청한다.

## 파일
1. C 프로그램에서 파일은왜필요할까?
  * 변수에 저장된 정보들은 실행이 끝나면 모두 사라진다. 
  * 정보를 영속적으로 저장하기 위해서는 파일에 저장해야 한다.
2. 유닉스 파일
  * 모든 데이터를 연속된 바이트 형태로 저장한다.
  * 키보드, 마우스도 파일로 인식하다.

## C 언어 파일 종류
1. 텍스트 파일
  * 사람들이읽을수있는문자들을저장하고있는파일
  * 텍스트파일에서“한줄의끝”을나타내는표현은파일이읽어들여질때, C 내부의 방식으로변환된다.
3. 이진 파일
  * 모든데이터는있는그대로바이트의연속으로저장
  * 이진파일을이용하여메모리에저장된변수값형태그대로파일에저장할수있다.

## 파일 입출력
1. C 언어의 파일입출력과정
   ```
  1. 파일 열기: fopen( ) 함수 사용
  2. 파일 입출력 : 다양한 파일 입출력함수사용
  3. 파일 닫기: fclose( ) 함수 사용
  ```

## 파일 열기
1. 파일을 사용하기 위해서는
  * 반드시먼저파일열기(fopen)를 해야한다.
  * 파일열기를하면FILE 구조체에대한포인터가리턴되며
  * FILE 포인터는 열린 파일을 나타낸다. 
2. 함수 fopen()
  * FILE *fopen(const char *filename, const char *mode); 
  * const char *filename: 파일명에 대한 포인터
  * const char *mode: 모드로 파일을 여는 형식

## 파일 닫기
1. 파일을열어서사용한후에는파일을닫아야한다.
  * int fclose(FILE *fp ); 
   fp는 fopen 함수에서 받았던 포인터
   닫기에성공하면0, 오류일때는EOF( -1)를 리턴한다.
  * 예시 : fclose(fp);

## 문자 단위 입출력
1. fgetc() 함수와 fputc() 함수
  * 파일에문자단위입출력을할수있다
2. int fgetc(FILE *fp);
  *  fp가 지정한파일에서한문자를읽어서리턴한다. 
   파일끝에도달했을경우에는EOF(-1)를리턴한다.
3. int fputc(int c, FILE *fp);
  *  fp가 가리키는파일에한문자씩출력하는함수
   리턴값으로출력하는문자리턴
   출력시오류가발생하면EOF(-1)리턴
